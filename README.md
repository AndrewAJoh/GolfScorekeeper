# PlusOne: Simple Golf Scorekeepe
Plus One is a golf scorekeeping app intended to be used while you golf. It is meant to be used in place of a physical score caddy. No sign-in or account required!

##Version 1.0 Features:
* Easily add your favorite golf courses by typing in a sample scorecard - 9 or 18 holes
* Track your score as you play a round of golf
* Automatically pause and resume golf round when you leave the app

##Version 1.1 Features (estimated release July 2021):
* Automatically saves round data after a user has completed round
* Increased readability through greater font sizes and ability to zoom in on scorecard
* Additional warnings to avoid accidentally losing data via incorrect button press
* Allowing user to play the round in any order they want rather than sequentially

The application can be downloaded to any Samsung Watch or Watch Active device (any version) via the Samsung Galaxy Store
